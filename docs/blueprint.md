# **App Name**: PlumbQuick

## Core Features:

- Service Listings: Display a list of plumbing services offered.
- Contact Form: Allow users to submit inquiries via a contact form.
- AI Issue Analyzer: An AI powered tool that provides preliminary diagnosis and advice based on user-submitted descriptions of their plumbing issues.
- Emergency Button: Provides easy contact access in urgent plumbing situations.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A), evoking reliability and professionalism.
- Background color: Light gray-blue (#F0F4F8), a muted, desaturated tint of the primary hue that provides a clean and trustworthy backdrop.
- Accent color: Warm orange (#E67700), for a high contrast highlight that calls attention to important actions and information.
- Headline font: 'Space Grotesk', a sans-serif, to convey a modern, computerized and efficient impression; body font: 'Inter', a sans-serif, for easy readability of longer text.
- Use clear, recognizable icons representing common plumbing services and issues.
- A clean, organized layout with clear sections for services, contact information, and issue diagnosis.
- Subtle animations for form interactions and page transitions.
